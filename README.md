# からだ施術マッチ (Wellness Matching App)

あなたの体の不調に寄り添い、最適な施術方法（整体、鍼灸、カイロ等）と専門のセラピストをマッチングするMVPアプリケーションです。

## 主な機能

- **不調チェック機能**: ユーザーの症状（肩こり、腰痛、眼精疲労など）をヒアリング
- **推薦アルゴリズム**: 症状に基づいたマッチ度計算とセラピストの推薦
- **セラピスト一覧・詳細**: 専門家のプロフィール、対応可能な不調、所在地などの確認
- **管理者ダッシュボード**: セラピスト情報の追加・編集・削除 (CRUD)
- **簡易認証**: 管理者向けのパスワード認証機能

## 技術スタック

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS / Lucide React (Icons)
- **Database**: SQLite
- **ORM**: Prisma

## セットアップ手順

1. **リポジトリのクローン/展開**
2. **依存関係のインストール**
   ```bash
   npm install
   ```
3. **環境変数の設定**
   `.env` ファイルを作成し、以下の項目を設定してください（デフォルト値も利用可能です）。
   ```env
   DATABASE_URL="file:./dev.db"
   ADMIN_PASSWORD="admin123"
   ```
4. **データベースの初期化とマイグレーション**
   ```bash
   npx prisma db push
   ```
5. **ダミーデータの投入 (Seed)**
   ```bash
   npx prisma db seed
   ```
   ※ `package.json` に設定されているため、上記コマンドで `prisma/seed.ts` が実行されます。
6. **開発サーバーの起動**
   ```bash
   npm run dev
   ```

## 管理画面へのアクセス

- URL: `/admin`
- デフォルトパスワード: `admin123`

## プロジェクト構造

- `src/app`: 各ページ、APIルート、サーバーアクション
- `src/components`: 共通コンポーネント、管理画面用コンポーネント
- `src/lib`: Prismaクライアントなどの共通ライブラリ
- `prisma`: データベーススキーマ、マイグレーション、シードデータ

## 注意事項

- 本アプリケーションはMVP（実証実験版）であり、実際の予約決済機能や高度なセキュリティ機能は含まれていません。
- 医療的な診断を目的としたものではなく、あくまでウェルネス向上のためのマッチングツールです。
